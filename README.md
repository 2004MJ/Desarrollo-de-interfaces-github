MiniWord
MiniWord es un editor de texto básico desarrollado en Python utilizando PySide6. Permite crear, abrir, guardar y editar archivos de texto con funcionalidades adicionales como contar palabras, buscar y reemplazar texto, y personalizar el color de fondo del editor.
Características
Crear, abrir y guardar archivos de texto (.txt).
Deshacer y rehacer cambios.
Copiar, cortar y pegar texto.
Contador de palabras en tiempo real.
Buscar y reemplazar texto (una coincidencia o todas las coincidencias).
Cambiar el color de fondo del editor.
Panel de búsqueda y reemplazo integrado.
Aviso de guardado al cerrar archivos sin guardar.
